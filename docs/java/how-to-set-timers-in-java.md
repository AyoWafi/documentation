To set a timer in Java, use [`sleep()`](https://www.javadoc.io/doc/io.temporal/temporal-sdk/latest/io/temporal/workflow/Workflow.html#sleep) and pass how many seconds you want to wait before continuing.

```java
sleep(5);
```
